# 杭州西湖陪拍师-宝哥人像摄影网站 - 开发文档

## 📋 项目概述

### 项目名称
杭州西湖陪拍师-宝哥 (人像摄影个人展示网站)

### 项目目标
构建一个专注于人像摄影作品展示的静态网站，采用现代化前端技术栈，强调视觉沉浸感与交互流畅性。整体设计遵循暗色主题，注重响应式体验与性能优化。

### 核心特点
- 暗色极简主题设计
- 瀑布流作品展示
- 全屏沉浸式画廊
- 滚动感知交互
- 响应式布局

- 组件间数据共享需求有限
- 简化架构降低复杂度

## 🔧 核心功能实现

### 1. 瀑布流布局

#### 实现方式
使用 CSS Grid 实现瀑布流布局，通过动态计算列数实现响应式。

#### 关键代码 (`WaterfallGallery.vue`)

```vue
<template>
  <div class="waterfall-gallery" :style="{ '--column-count': columnCount }">
    <div
      v-for="photo in photos"
      :key="photo.id"
      class="gallery-item"
      :style="{ '--aspect-ratio': photo.aspectRatio }"
    >
      <img :src="photo.thumbnail" :alt="photo.title" loading="lazy" />
    </div>
  </div>
</template>

<script setup>
const columnCount = computed(() => {
  if (window.innerWidth < 768) return 2
  if (window.innerWidth < 1024) return 3
  return 4
})
</script>

<style>
.waterfall-gallery {
  display: grid;
  grid-template-columns: repeat(var(--column-count), 1fr);
  gap: 16px;
}
</style>
```

#### 图片宽高比
每个图片项通过 `aspect-ratio` CSS 属性保持原始宽高比:
```css
.gallery-item {
  aspect-ratio: var(--aspect-ratio);
}
```

### 2. 画廊页交互

#### 单击返回
实现单击图片返回首页的逻辑，需要区分单击和双击:

```javascript
const lastClickTime = ref(0)

const handleImageClick = () => {
  const now = Date.now()
  if (now - lastClickTime.value < 300) {
    // 双击，忽略单击
    return
  }

  // 单击，返回首页
  setTimeout(() => {
    router.push('/')
  }, 300)
}

const handleImageDoubleClick = () => {
  isZoomed.value = !isZoomed.value
  lastClickTime.value = Date.now()
}
```

#### 双击放大
通过切换 `isZoomed` 状态控制图片缩放:

```vue
<template>
  <img
    :class="{ zoomed: isZoomed }"
    @dblclick="handleImageDoubleClick"
  />
</template>

<style>
.gallery-image.zoomed {
  max-width: 150%;
  max-height: 150vh;
}
</style>
```

#### ESC 退出
监听键盘事件，支持 ESC 键快速退出:

```javascript
onMounted(() => {
  const handleKeyDown = (e) => {
    if (e.key === 'Escape') {
      router.push('/')
    }
  }
  window.addEventListener('keydown', handleKeyDown)
})

onUnmounted(() => {
  window.removeEventListener('keydown', handleKeyDown)
})
```

### 3. 滚动感知悬浮按钮

#### 实现原理
使用 `requestAnimationFrame` 优化滚动事件处理，检测滚动方向:

```javascript
// composables/useScrollDirection.js
export function useScrollDirection(threshold = 100) {
  const isHidden = ref(false)
  const lastScrollY = ref(0)
  const ticking = ref(false)

  const handleScroll = () => {
    if (!ticking.value) {
      window.requestAnimationFrame(() => {
        const currentScrollY = window.scrollY

        if (currentScrollY > lastScrollY.value && currentScrollY > threshold) {
          isHidden.value = true  // 向下滚动隐藏
        } else if (currentScrollY < lastScrollY.value) {
          isHidden.value = false // 向上滚动显示
        }

        lastScrollY.value = currentScrollY
        ticking.value = false
      })
      ticking.value = true
    }
  }

  return { isHidden }
}
```

#### CSS 动画
使用 CSS `transition` 实现平滑的淡入淡出效果:

```css
.contact-button {
  transition: all 0.3s ease;
}

.contact-button.hidden {
  opacity: 0;
  transform: translateY(-20px);
  pointer-events: none;
}
```

### 4. 图片懒加载

#### 原生懒加载
使用 HTML5 `loading="lazy"` 属性:

```html
<img src="image.jpg" alt="Description" loading="lazy" />
```

#### 优点
- 原生支持，无需额外库
- 浏览器优化，性能好
- 简单易用

#### 加载状态
在画廊页显示加载动画:

```vue
<template>
  <div v-if="isLoading" class="loading-overlay">
    <div class="loading-spinner"></div>
  </div>
</template>

<style>
.loading-spinner {
  border: 4px solid var(--border-color);
  border-top-color: var(--text-primary);
  border-radius: 50%;
  animation: spin 1s linear infinite;
}

@keyframes spin {
  to { transform: rotate(360deg); }
}
</style>
```

## 📦 资源管理

### 图片目录结构

```
src/assets/images/
├── thumbnails/     # 缩略图 (<150KB)
│   ├── photo1.webp
│   ├── photo2.webp
│   └── ...
└── originals/      # 原图 (高清)
    ├── photo1.jpg
    ├── photo2.jpg
    └── ...
```

### 图片优化建议

#### 缩略图
- **格式**: WebP (优先) 或 JPEG
- **尺寸**: 短边 800px
- **质量**: 70%
- **大小**: <150KB
- **工具**: TinyPNG, Squoosh, sharp

#### 原图
- **格式**: JPEG
- **保留**: 原始分辨率和质量
- **加载**: 仅在画廊页按需加载
