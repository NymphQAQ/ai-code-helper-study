
项目名称  
杭州西湖陪拍师-宝哥(人像摄影个人展示网站)

一、项目概述  
构建一个专注于人像摄影作品展示的静态网站，采用现代化前端技术栈，强调视觉沉浸感与交互流畅性。整体设计遵循暗色主题，注重响应式体验与性能优化。

二、技术选型  
框架：Vue 3（Composition API）  
构建工具：Vite  
部署方式：静态站点（可部署于 Vercel / Netlify / GitHub Pages 等）  
图片处理：前端懒加载 + 响应式压缩图（原图仅在画廊页加载）

三、页面结构  

首页（Home）  
布局：瀑布流（Masonry Layout）  
内容：展示人像摄影作品缩略图（已压缩，兼顾加载速度与视觉质量）  
交互：
  点击任意缩略图 → 跳转至对应作品的沉浸式画廊页
响应式：适配移动端、平板、桌面端

画廊页（Gallery / Detail）  
布局：全屏沉浸式视图，隐藏导航及其他 UI 元素  
功能：
  默认显示高清原图（支持高分辨率）
  单击图片：返回首页（带过渡动画）
  双击图片：放大查看细节（可配合 pinch-to-zoom 或 modal zoom 实现）
交互反馈：平滑缩放动画、加载状态提示

关于页（About）  
内容：摄影师简介、创作理念、联系方式等  
风格：延续暗色主题，排版简洁优雅  
访问路径：通过主导航或页脚链接进入（首页不直接展示 About 入口）

四、UI/UX 设计规范  

主题风格  
主色调：深灰/黑色背景（#0f0f0f 或类似）  
文字/控件：浅色系（如 #ffffff / #e0e0e0），低对比度以减少视觉疲劳  
整体风格：极简、现代、聚焦内容本身

首页右上角悬浮按钮（Contact / Social）  
功能：点击后弹出模态框，展示微信 & 抖音二维码图片  
视觉要求：
  图标需与暗色背景自然融合（如使用半透明白色线性图标）
  不突兀，保持界面干净
交互逻辑：
  页面初始加载时显示
  用户向下滚动时淡出隐藏
  向上滚动时淡入显示
  动画：使用 CSS transition 实现平滑淡入淡出（opacity + transform）
弹窗样式：
  模态框居中，背景半透明遮罩
  两个二维码并列展示，标注平台名称
  点击遮罩或关闭按钮可关闭

五、性能与体验要求  
所有首页图片必须为压缩后的 WebP/JPEG（建议 <150KB）
画廊页按需加载原图，避免首页资源过载
支持 PWA 基础能力（可选）
无障碍基础支持（alt 文本、键盘导航等）

六、交付物清单  
完整 Vue + Vite 项目源码  
响应式瀑布流组件（可基于 vue-waterfall-plugin 或自研）  
画廊页交互逻辑（含单击返回、双击放大）  
滚动感知的悬浮按钮组件（含淡入淡出逻辑）  
二维码弹窗组件  
静态资源目录（含示例图片占位）  
README.md（含本地运行与部署说明）
