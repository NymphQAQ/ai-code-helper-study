## 🎯 项目概述

这是一个专注于人像摄影作品展示的静态网站，面向杭州西湖陪拍师"宝哥"的个人品牌展示。网站采用现代化前端技术栈，强调视觉沉浸感与交互流畅性，整体设计遵循暗色主题，注重响应式体验与性能优化。

---

## ✨ 核心功能

### 1. 首页 (Home)
- **瀑布流布局**: 使用 CSS Grid 实现响应式瀑布流展示
- **图片懒加载**: 优化首屏加载速度
- **点击交互**: 点击缩略图跳转至画廊页
- **视觉效果**: 悬停动画、渐进式加载

### 2. 画廊页 (Gallery)
- **全屏沉浸式视图**: 专注于作品展示
- **单击返回**: 单击图片返回首页 (带过渡动画)
- **双击放大**: 双击图片放大查看细节
- **ESC 退出**: 支持键盘快捷键退出
- **加载状态**: 显示加载动画

### 3. 联系功能
- **悬浮按钮**: 右上角固定位置
- **滚动感知**: 向下滚动隐藏，向上滚动显示
- **二维码弹窗**: 展示微信 & 抖音二维码
- **平滑动画**: 淡入淡出过渡效果

### 4. 关于页 (About)
- 摄影师简介
- 创作理念
- 服务内容列表
- 联系方式展示

## 🔧 关键实现

### 1. 瀑布流布局 (CSS Grid)

使用 CSS Grid 和 `aspect-ratio` 属性实现自适应瀑布流，无需第三方库。

### 2. 单双击事件区分

精确区分单击和双击事件，避免冲突:
```javascript
const handleImageClick = () => {
  const now = Date.now()
  if (now - lastClickTime.value < 300) return // 双击已处理
  // 单击逻辑
}

const handleImageDoubleClick = () => {
  isZoomed.value = !isZoomed.value
  lastClickTime.value = Date.now()
}
```

### 3. 滚动优化

使用 `requestAnimationFrame` 优化滚动事件处理，提高性能:
```javascript
const handleScroll = () => {
  if (!ticking.value) {
    window.requestAnimationFrame(() => {
      // 处理逻辑
      ticking.value = false
    })
    ticking.value = true
  }
}
```

### 4. 原生懒加载

使用 HTML5 `loading="lazy"` 属性实现图片懒加载:
```html
<img src="image.jpg" alt="Description" loading="lazy" />
```

## 📊 性能指标

### 优化措施

1. **图片优化**:
   - 缩略图使用 WebP 格式
   - 压缩至 <150KB
   - 原图仅在画廊页按需加载

2. **代码优化**:
   - 路由级代码分割
   - Tree Shaking
   - 生产构建压缩

3. **加载优化**:
   - 图片懒加载
   - 异步组件加载
   - 预加载关键资源

- 首页加载时间: <2秒 (3G网络)
- 首页图片总大小: <1.5MB
