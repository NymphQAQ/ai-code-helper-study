## ⚠️ 重要：图片压缩步骤

您的原始照片文件非常大（每张 10-25MB），**必须压缩后才能使用**！

### 快速开始（推荐在线工具）

#### 方法 1: TinyPNG (最简单) ⭐
1. 访问 [https://tinypng.com/](https://tinypng.com/)
2. 拖拽前 12 张照片上传（DSC05537.JPG ~ DSC05552.JPG）
3. 等待压缩完成
4. 下载并解压
5. 将压缩后的图片放入 `public/thumbnails/` 目录

#### 方法 2: 使用脚本 (需要安装 ImageMagick)
```bash
# 安装 ImageMagick
brew install imagemagick

# 运行压缩脚本
./compress-images.sh
```

**详细指南**: 请查看 [图片压缩指南.md](./图片压缩指南.md)

## 📝 接下来的操作

### 1. 更新联系方式

编辑文件: `src/components/ContactModal.vue`

替换二维码图片和文字信息:
```vue
<div class="qr-placeholder">
  <img src="/your-wechat-qr.png" alt="微信二维码">
</div>
```

### 2. 自定义作品标题

编辑文件: `src/components/WaterfallGallery.vue`

找到 `photos` 数组，修改 `title` 字段:
```javascript
{
  id: 1,
  title: '您的作品标题',  // 修改这里
  thumbnail: '/placeholder/DSC05537.JPG',
  original: '/placeholder/DSC05537.JPG',
  aspectRatio: '3/2'
}
```

### 3. 添加更多作品

在 `photos` 数组中添加更多对象:
```javascript
{
  id: 13,
  title: '新作品',
  thumbnail: '/placeholder/DSC05553.JPG',
  original: '/placeholder/DSC05553.JPG',
  aspectRatio: '3/2'
}
```

## ❓ 常见问题

### Q: 网站加载很慢？
**A**: 这是因为原始照片文件太大（10-25MB/张）。请务必按照 [图片压缩指南](./图片压缩指南.md) 压缩图片后再使用。

### Q: 如何替换二维码？
**A**: 编辑 `src/components/ContactModal.vue`，将占位图替换为实际的二维码图片路径。

### Q: 如何修改网站标题？
**A**: 编辑 `index.html` 中的 `<title>` 和 `<meta name="description">` 标签。

### Q: 如何添加更多页面？
**A**: 
1. 在 `src/views/` 创建新组件
2. 在 `src/main.js` 中添加路由配置

### Q: 如何绑定自己的域名？
**A**: 
1. 购买域名
2. 在 Vercel/Netlify 控制台添加域名
3. 在域名服务商添加 DNS 解析记录
