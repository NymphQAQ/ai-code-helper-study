# AI Code Helper - 接口文档

> 版本：1.0.0 | 更新日期：2026-03-08

---

## 概述

本项目后端基于 Spring Boot 4，context-path 为 `/api`，所有接口均以 `/api` 为前缀。  
核心通信协议为 **SSE（Server-Sent Events）**，前端通过浏览器原生 `EventSource` 接收流式响应。

**基础地址**

| 环境 | 地址 |
|------|------|
| 本地开发 | `http://localhost:8080/api` |
| 前端代理后 | `http://localhost:5173/api`（Vite proxy 转发） |

---

## 通用说明

### 跨域（CORS）

后端已全局配置 CORS，允许所有来源、所有方法、所有请求头，并允许携带凭证。  
生产环境建议将 `allowedOriginPatterns` 收窄为前端真实域名。

### 错误处理

当 `SafeInputGuardrail` 拦截到敏感词时，后端会抛出异常，SSE 连接的 `onerror` 事件将被触发。  
前端应在 `onError` 回调中展示友好提示。

## 接口列表

### 1. 流式对话接口

#### 接口信息

| 属性 | 值 |
|------|-----|
| URL | `GET /api/ai/chat` |
| 协议 | SSE（Server-Sent Events） |
| 认证 | 无（内网使用，生产环境建议添加 Token） |
| Content-Type | `text/event-stream` |

#### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `memoryId` | `int` | 是 | 会话 ID，相同 ID 共享对话历史（最多 10 条），不同 ID 独立隔离 |
| `message` | `string` | 是 | 用户输入的消息内容，需 URL 编码 |

#### 请求示例

```
GET /api/ai/chat?memoryId=1&message=%E4%BB%8B%E7%BB%8D%E4%B8%80%E4%B8%8B%E8%87%AA%E5%B7%B1
```

等价原始请求：
```
GET /api/ai/chat?memoryId=1&message=介绍一下自己
```

#### 响应格式

响应为 SSE 事件流，每个事件格式如下：

```
data: <token文本片段>


```

完整响应示例（多个事件组成完整回答）：

```
data: 你好

data: ！我是

data: AI 代码助手

data: ，有什么可以帮你的呢？

```

当流结束时，连接关闭（`readyState === EventSource.CLOSED`），前端 `onerror` 被触发，  
此时 `readyState === 2` 表示正常结束，非错误状态。

#### 前端调用示例

```javascript
// 使用项目封装的 chatStream 函数
import { chatStream } from './api/chat.js'

chatStream(1, '介绍一下自己', {
  onMessage: (data) => {
    console.log('收到 token：', data)
  },
  onError: (error) => {
    console.error('发生错误：', error)
  },
  onComplete: () => {
    console.log('响应完成')
  }
})
```

也可以直接使用原生 EventSource：

```javascript
const url = '/api/ai/chat?memoryId=1&message=' + encodeURIComponent('你好')
const es = new EventSource(url)

es.onmessage = (event) => {
  console.log(event.data)
}

es.onerror = () => {
  es.close()
}
```

#### 错误情形

| 情形 | 表现 | 原因 |
|------|------|------|
| 消息包含敏感词 | SSE 连接立即关闭，触发 onerror | SafeInputGuardrail 拦截 |
| API Key 无效 | SSE 连接关闭，触发 onerror | DashScope 认证失败 |
| 后端未启动 | EventSource 无法建立连接 | 网络/服务异常 |

## 前端 API 模块说明

文件路径：`front/src/api/chat.js`

### chatStream(memoryId, message, callbacks)

SSE 流式请求，立即返回结果。

| 参数 | 类型 | 说明 |
|------|------|------|
| `memoryId` | number | 会话 ID |
| `message` | string | 用户消息 |
| `callbacks.onMessage` | function(data) | 每收到一个 token 时调用 |
| `callbacks.onError` | function(error) | 发生错误时调用 |
| `callbacks.onComplete` | function() | 流式响应完成时调用 |

**返回值**：取消函数 `() => void`，调用后关闭 EventSource。

### chat(memoryId, message)

Promise 方式，收集全部 token 后一次返回。

| 参数 | 类型 | 说明 |
|------|------|------|
| `memoryId` | number | 会话 ID |
| `message` | string | 用户消息 |

**返回值**：`Promise<string>`，完整 AI 回复文本。
